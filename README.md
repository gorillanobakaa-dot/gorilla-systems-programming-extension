# Gorilla Systems Programming Extension (Gorilla Edition) 🦍

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Gorilla Systems** is a high-performance architectural framework for the Gemini CLI. It transforms your AI assistant into a **Master Orchestration Agent** capable of managing massive, complex codebases and specialized build environments.

---

## 🌟 What is it?
This extension is designed for developers who handle large-scale systems engineering tasks. It doesn't just provide tools; it provides a **senior collective of specialized sub-agents** that work in parallel to monitor, debug, and optimize your software builds.

### Core Focus:
- **Firefox 150+ Ecosystem**: Deep integration with Gecko, SpiderMonkey, and `mozbuild`.
- **Cross-Compilation Mastery**: Expertly targets Windows 10/11 (MSVC) and legacy macOS from Linux hosts.
- **Systematic Debugging**: A rigorous protocol for isolating failures and deploying surgical fixes.

---

## 🛠️ The Agent Roster
When you activate Gorilla Systems, you get access to a team of digital experts:
- 🏗️ **@build_engineer**: The toolchain and configuration architect.
- 🕵️ **@codebase_investigator**: Analyzes C++ and Rust syntax/linkage.
- 📚 **@dependency_resolver**: Manages sysroots and SDKs.
- 🩺 **@runtime_debugger**: Triages crashes and system blocks.
- 🛡️ **@security_auditor**: Scans for overflows and race conditions.
- 🏁 **@quality_supervisor**: Ensures code sanity and prevents hallucinations.
- 📂 **@custodian_archivist**: Maintains a persistent record of the entire process.

---

## 🚀 One-Click Installation
If you have the Gemini CLI installed, simply run this command in your terminal:

```bash
gemini extensions install https://github.com/gorillanobakaa-dot/gorilla-systems-programming-extension
```
